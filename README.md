客人 :

下訂單
查看訂單

查看排班表

廚師  : 
查看排班表  
查看訂單

--------輸入與輸出資料分析-----------
下訂單(客人、會員)  : 餐點,桌號  -> 訂單內容
查看訂單(客人、會員)  : 編號/時間/桌號 -> 訂單內容
註冊(客人)  : 帳號,密碼,使用者名稱  -> ID
查看歷史訂單 (會員) : 時間 ->訂單內容

查看排班表(員工)  : 時間/編號  -> 員工姓名、編號、上班時間
查詢桌位(服務員) :  輸出桌號、是否空桌
查看食材  : 食材名稱 -> 食材數量
查看全部食材 :  輸出食材名稱、數量

訂單內容  : 編號、餐點、時間、金額、分店、桌號、付款方式、折扣
